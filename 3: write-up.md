# Design Rationale

**Questions**: Non-leading, psych-based (Rotter Locus, Dweck Mindset Axis1; Campbell/Organ Axis2; Maslow/Batson Axis3). 4-5 opts balanced.

**Branching**: Decisions on answers/signals; bridges link axes progressively.

**Trade-offs**: 30 nodes for depth, simple TSV format.

**Sources**: Rotter 1954, Dweck 2006, Campbell 2004, Organ 1988, Maslow 1969.

**AI Guardrails**: Perplexity drafts → manual 4-persona tests.

**Improvements**: User feedback loops, mobile.
